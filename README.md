# react-native-audio-gphone

audio socket

## Installation

```sh
npm install react-native-audio-gphone
```

## Usage

```js
import { initPlayAudio,writePlayAudio } from 'react-native-audio-gphone';

// ...

initPlayAudio();
writePlayAudio(BASE64);

```
